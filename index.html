Duvidas Frequentes

1. Quanto tempo leva para desenvolver um sistema?
depende muito do tipo de sistema que o cliente está adquirindo, landing pages por exemplo, são menos complexas de criar, do que um site que demanda muitas outras ferramentas e questões estruturais.
2. A Code oferece suporte após a entrega do sistema?
Sim, quando o sistema é entregue, o cliente recebe um treinamento para se familiarizar com o sistema. Qualquer duvida em relação ao funcionamento, duvidas ou bugs podem ser sanados no prazo de 90 dias após a entrega do site de forma gratuita (sem serviço recorrente). Para obter mais tempo de suporte a mais tempo e manutenções ou personalizações, somente com o serviço de recorrência.
3. Meu sistema vai funcionar em celular e tablet?
sim, todos os nossos sistemas são compatíveis e responsivos para se adequar tanto no funcionamento, quanto na experiência visual do usuário aos aparelhos celulares e tablets.
4. Posso sugerir mudanças após a entrega do meu sistema?
Sim pode, mas esse processo só é possível caso o usuário tenha acesso ao plano de serviço recorrente, caso o cliente não tenha, não é possível fazer manutenções ou alterações. (ver condições de alterações).
5. A Code oferece hospedagem ou preciso contratar fora à parte?
Sim, a hospedagem de onde seu site funcionará, já é incluso ao fechar um projeto.
6. Como é feito o orçamento?
O cliente poderá deixar seus dados para que a nossa equipe entre em contato com o mesmo, para marcar uma reunião e conversar sobre qual produto o cliente pretende investir, ou clicando aqui (link whatsapp) para ser redirecionado e marcar um data para uma reunião.
